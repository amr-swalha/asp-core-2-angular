Components

Are the building block of any angular app. You define your view and code behind on them.

You can define a component by adding a ts file to the shared module

Component Configuration

  • templateUrl: Where is the view html file is located
  • selector: the html tag name for the component
  • styles: The css for th ecomponent
  • styleUrls: the css file for the component

Component Event Life Cycle

It's where you can hook up to the event life cycles of the components

  • ngOnInit: on calling the component
  • ngOnDestroy: when the component is destroyed
  • ngAfterViewInit: after the view has been initialized

Component Reference

You can call any component using it's selector
