Data binding

Databinding is used to pass data from the component class to the view.

There are three way of data binidng:

  • One Way -> From data source to view. Syntax: [target]
  • One Way -> From view to data source. Syntax: (event)
  • Two Way -> From Datasource to view. Syntax: [(model)]
